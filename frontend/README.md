# AI Photobooth Frontend

Frontend React cho ứng dụng AI Photobooth.

## Cài đặt

```bash
npm install
```

## Chạy development

```bash
npm run dev
```

Ứng dụng sẽ chạy tại http://localhost:3000

## Build production

```bash
npm run build
```

Output sẽ nằm trong thư mục `dist/`

## Preview production build

```bash
npm run preview
```

## Công nghệ sử dụng

- React 18
- Vite
- TailwindCSS
- Lucide Icons
- Axios
